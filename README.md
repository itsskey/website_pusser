# БОКС НАЧНИ — лендинг

Одностраничный лендинг с кнопкой призыва к действию, ведущей на внешнюю страницу оплаты/регистрации.

## Структура проекта

- `index.html` — разметка страницы (заголовки, кнопка, подвал).
- `style.css` — оформление: черный фон, контрастные заголовки, стили кнопки и футера. 
- `script.js` — логика кнопки, перенаправление пользователя на внешний URL при клике. 

## Функциональность

- Отображение промо-блока с заголовками и описанием. 
- Кнопка «Начать» (или аналогичная), расположенная по центру, оформлена в фирменном стиле (желтый фон, скругленные края, ховер-эффект). 
- При клике на кнопку элемент с `id="start-box"` открывает внешний URL через `window.location.href`.

## Установка и запуск

1. Склонируйте репозиторий или скачайте архив с файлами. 
2. Убедитесь, что `index.html`, `style.css` и `script.js` находятся в одной директории.
3. Откройте `index.html` двойным кликом или через локальный веб-сервер.

## Кастомизация

- Текст заголовков и описания меняется в `index.html`.
- Цвета, шрифты, размеры кнопки и заголовков настраиваются в `style.css`.
- Целевой URL для перехода меняется в `script.js` в значении `window.location.href`.

## Структура HTML (кратко)

- `header` с основным оффером и кнопкой.
- `footer` с дополнительной информацией/ссылками (например, политика конфиденциальности, оферта).
