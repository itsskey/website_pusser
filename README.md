# БОКС НАЧНИ — лендинг

Одностраничный лендинг с кнопкой призыва к действию, ведущей на внешнюю страницу оплаты/регистрации. [file:1][file:2]

## Структура проекта

- `index.html` — разметка страницы (заголовки, кнопка, подвал). [file:1]
- `style.css` — оформление: черный фон, контрастные заголовки, стили кнопки и футера. [file:3]
- `script.js` — логика кнопки, перенаправление пользователя на внешний URL при клике. [file:2]

## Функциональность

- Отображение промо-блока с заголовками и описанием. [file:1]
- Кнопка «Начать» (или аналогичная), расположенная по центру, оформлена в фирменном стиле (желтый фон, скругленные края, ховер-эффект). [file:1][file:3]
- При клике на кнопку элемент с `id="start-box"` открывает внешний URL через `window.location.href`. [file:2]

## Установка и запуск

1. Склонируйте репозиторий или скачайте архив с файлами. [file:1]
2. Убедитесь, что `index.html`, `style.css` и `script.js` находятся в одной директории. [file:1][file:3]
3. Откройте `index.html` двойным кликом или через локальный веб-сервер. [file:1]

## Кастомизация

- Текст заголовков и описания меняется в `index.html`. [file:1]
- Цвета, шрифты, размеры кнопки и заголовков настраиваются в `style.css`. [file:3]
- Целевой URL для перехода меняется в `script.js` в значении `window.location.href`. [file:2]

## Структура HTML (кратко)

- `header` с основным оффером и кнопкой. [file:1]
- `footer` с дополнительной информацией/ссылками (например, политика конфиденциальности, оферта). [file:1][file:3
